Overview

The Zingle iOS SDK lets you bring conversations right into your app.

The Zingle class initializes the SDK, and shows the conversation.

An ZNGSettings object is passed to the initWithSettings:completionHandler: method to apply configuration options such as the appId, accent color, and notification display time.

The ZNGUser class is used to attach custom user properties to each conversation.

The ZNGConversation class allows you to send and retrieve messages, as well as set a delegate to respond to important events.

The ZNGMessage class provides information about each message in the conversation, including its text, the author’s name, and more.
